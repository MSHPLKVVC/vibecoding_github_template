# Workflow: Landing Creation

## What this file is

This workflow helps create landing pages using marketing, product, UX, UI, and frontend perspectives.

## When to use

Use this workflow when creating or improving a landing page.

## Active agents

- marketer
- product-manager
- ux-researcher
- ui-designer
- developer
- qa

## Steps

1. Marketer defines audience, pain, positioning, offer, and messaging.
2. Product Manager defines landing goal and conversion action.
3. UX Researcher reviews user journey and friction.
4. UI Designer creates page structure and section layout.
5. Developer implements the landing.
6. QA checks responsiveness, forms, tracking, and content consistency.

## Outputs

- landing strategy;
- page structure;
- copy outline;
- UI structure;
- implementation;
- QA checklist.
