# Decisions

## What this file is

This is the decision log for the project.

## Why this file exists

AI agents should not re-discuss important decisions every session.

Write decisions here so the project has memory.

## How to use

Add entries whenever a meaningful decision is made.

## Template

### YYYY-MM-DD — Decision title

#### Decision

What was decided?

#### Reason

Why was this decision made?

#### Alternatives considered

What other options were considered?

#### Status

Proposed / Accepted / Rejected / Deprecated

#### Impact

What parts of the project does this affect?
