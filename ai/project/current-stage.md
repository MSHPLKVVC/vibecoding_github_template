# Current Stage

## What this file is

This file tells the AI what stage the project is currently in.

## Why this file exists

It prevents the system from jumping too early into development.

## Common stages

- intake
- market-discovery
- product-discovery
- mvp-definition
- technical-planning
- development
- qa-release
- growth

## Stage

intake

## Active Agents

- orchestrator

## Objective

Clarify the project, audience, goals, constraints, and next stage.

## Exit Criteria

- project name is defined;
- business goal is clear;
- initial audience is described;
- products are listed;
- next stage is selected.
