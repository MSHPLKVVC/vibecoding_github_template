# Context

## What this file is

This file stores the broad project context.

It should explain what the project is, why it matters, who it serves, and what is known so far.

## Why this file exists

AI agents need persistent context across sessions.

Instead of keeping important context in chat history, store it here.

## What to write here

- Business background.
- Target users.
- Existing processes.
- Known problems.
- Product idea.
- Important constraints.
- Notes from meetings or discovery.
- Any domain knowledge the AI should know.
